# License-plate-recognition
This project aims to extract digits (0-9) and represent Arabic text as 'T' from license plate images. The pipeline involves preprocessing, character segmentation, and Optical Character Recognition (OCR) to isolate and recognize the characters on the license plate.

Features
Extract digits (0-9) and Arabic text from license plate images.
Use preprocessing techniques to clean and enhance image quality.
Use OCR to recognize digits and Arabic text as 'T'.
Customizable and extendable for different types of license plate formats.
